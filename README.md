# Totoro’s Japanese Translation  
## Totoro的日语翻译  
## トトロの日本語翻訳

Totoro’s Japanese Translation is a lifelong, bilingual translation archive.  
Totoro的日语翻译是一个终身维护的双语翻译档案。  
トトロの日本語翻訳は、生涯にわたる二言語の翻訳アーカイブです。

Here you will find books, essays, and the 100分de名著 series translated by me,  
displayed in a Japanese (or other source language)–Chinese side-by-side format.  
在这里，你可以找到我翻译的书籍、论文，以及《100分de名著》系列，  
以日语（或其他原语言）–中文的对照形式呈现。  
ここでは、私が翻訳した書籍、エッセイ、そして「100分de名著」シリーズを  
日本語（または他の原語）–中国語の対訳形式でご覧いただけます。

This project will be maintained for life.  
All translations are shared freely and not for profit.  
You may modify them based on this work, but you must indicate the source, and they must never be used for profit.  
这个项目将终身维护。  
所有翻译作品都是免费共享的，不能用于盈利。  
你可以在此基础上进行修改，但必须注明来源，且永远不可用于盈利。  
このプロジェクトは一生維持されます。  
すべての翻訳は自由に共有され、営利目的で使用することはできません。  
この作品を基に改変することは可能ですが、必ず出典を明示し、営利目的で使用することはできません。

I do not translate because I am gifted,  
but because I cannot abandon the act of understanding.  
我翻译，不是因为我天赋异禀，  
而是因为我无法放弃理解的行为。  
私は翻訳の才能があるからではなく、  
理解するという行為を放棄できないから翻訳しています。
