# Totoro的日语翻译 - 字幕排版与校对规范 (针对100分de名著)

## 1. 核心理念（Totoro精神）
“我翻译不是因为天赋异禀，而是因为无法放弃理解的行为。”
- **原则**：绝对尊重原意。原译文只要逻辑清晰、未偏离原意，禁止强行“拔高文笔”或为了炫技而大面积重写。
- **最小干预**：只修硬伤，不改个人翻译习惯，绝对禁止擅自修改时间轴和结构。

## 2. 标点与排版规范
- **中文标点**：全篇中文标点统一使用全角符号（，。！？）。
- **省略号**：无论是原文还是译文，省略号必须使用标准的“……”（六个点），禁止使用半角“...”。
- **破折号**：使用标准的“——”。
- **语气词控制**：日文演播室对谈中的“ね、よ、なぁ”等，翻译时需“断舍离”。禁止擅自在中文里强加没有对应含义的“啊、啦、呢、吧”，保持文字干净。

## 3. 双轨语境分离（重中之重）
《100分de名著》同时包含“学术引用”与“通俗解说”，必须呈现两种截然不同的文风：
- **【名著朗读 / 引用段落】**：要求古典、严谨、庄重，尽可能向中文出版界的“通行译法”（或名家译本）靠拢。
- **【演播室对谈 / 嘉宾解说】**：要求口语化、接地气。遇到抽象的哲学概念，嘉宾常用比喻，必须把比喻翻译得生动自然。

## 4. 逻辑与STT纠偏指南
- **填补主语**：日文常省略主语，中文翻译时必须根据语境（尤其是尊他/自谦动词）准确还原“谁对谁做某事”。
- **条件与转折**：重点排查条件句（たら/れば）是否漏翻，转折词（が/けど/のに）是否被误译为顺承。
- **STT同音词幻觉**：字幕原本若是AI语音识别生成，遇到句子讲不通时，必须尝试用日文罗马音反推同音词（如把“間暇”纠正为“閑暇”）。
